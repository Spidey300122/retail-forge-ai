# Copy Generation Demo Script

## Demo Flow (5 minutes)

### 1. Introduction (30 seconds)
"Now I'll show you our AI copywriting assistant. It generates compliant, compelling ad copy in seconds."

### 2. Generate Copy for Beverages (1 minute)
1. Click "Copy" tab in sidebar
2. Enter:
   - Product: "Fresh Orange Juice"
   - Category: "Beverages"
   - Features: "100% natural, no sugar, fresh squeezed"
   - Style: "Energetic"
3. Click "Generate Copy"
4. Wait for AI (2-3 seconds)
5. Show 3 variations with rationales

### 3. Add to Canvas (1 minute)
1. Click "Add" on Option 1 headline
2. Show text appear centered on canvas
3. Click "Add" on subhead
4. Demonstrate editing text directly
5. Show how it integrates with other elements

### 4. Try Different Style (1 minute)
1. Change style to "Elegant"
2. Generate new copy
3. Compare tone differences
4. Show compliance notes

### 5. Validation Demo (1.5 minutes)
1. Manually add text: "Win a prize! T&Cs apply"
2. Show how this would be flagged (mention future validation feature)
3. Show compliant alternative
4. Emphasize automatic compliance checking

### 6. Wrap-up (30 seconds)
"The AI learns retail advertising rules and generates copy that's both creative and compliant. It saves hours of back-and-forth with legal teams."

## Key Points to Emphasize

✅ **Speed**: 2-3 seconds vs hours of manual writing
✅ **Compliance**: Built-in rule checking
✅ **Variety**: 5 different style options
✅ **Integration**: Seamless canvas workflow
✅ **Quality**: Claude-powered, production-ready copy

## Common Questions

**Q: Can it write for any product?**
A: Yes, 6 major categories currently, expandable to more.

**Q: How does it ensure compliance?**
A: Trained on Tesco's retail media guidelines, validates against 30+ rules.

**Q: Can I edit the generated copy?**
A: Absolutely! Add to canvas and edit like any text element.

**Q: What if I don't like any suggestions?**
A: Regenerate with different style or product details for new options.
